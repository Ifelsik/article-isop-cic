# Модуль для генерации тестовых данных

Python-модуль, который позволяет генерировать тестовые входные данные в папку data
в корне проекта.

Кроме того, модуль отрисовывает графики для входных и теоретических выходных.

Предварительно надо установить зависимости проекта:
```
pip install -r requirements.txt
```
